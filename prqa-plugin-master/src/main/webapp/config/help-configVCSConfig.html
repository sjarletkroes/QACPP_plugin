Specify the full path of the version control system configuration file, which must have an .xml extension.
The QA·Verify Client package comes with a number of configuration files to support a wide range of Version Control Systems. These can be found in the VCS sub directory of the QA·Verifty Client installation. Note that the configuration file may need to be customised to work with particular deployments of the Version Control System.
If you have chosen All code uploaded above you can still select a configuration file to include the file versions and authors in the snapshot. If you do not want the file information you should choose single-sop.xml.
